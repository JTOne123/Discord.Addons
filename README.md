# EmbedOnFly
Faster embeds for Discord.NEt 1.0+ warapper

Example Usage :

```cpp
Embed embed = new Embed();

embed.MakeEmbed("red", "Example", "this is example");
embed.SendEmbed(Context);
```

Colors supported :

Aqua | Black | Blue
Fuchsia | Gray | Green
Lime | Maroon | Navy
Olive | Purple | Red
Silver | Teal | White
Yellow | / | /
